Store Inventory Manager

This is a program that helps users store items and save it in their local browser storage.

Let's Inventory

Add an item to your inventory by filling out the form below, then click the Add button.

Code Item Description Qty Actions

Instructions:

Adding an item: Fill in the code, name, description, and quantity. Then click Add.

Edit an item: Click Edit beside the item you wish to modify.

Deleting an item: Click Delete beside the item you wish to remove.

Clear Data: Click Settings then Clear Data to wipe everything.

Custom Name: Click Settings then Edit Title to rename your inventory.

About this program:

This program helps you track store items. It records, modifies, and deletes items, all saved in your browser.

Data stays until you clear your browser storage.
